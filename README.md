# RezeptManager

Fullstack recipe management application built with **React and Spring Boot**.

The project demonstrates a practical CRUD workflow combined with **OCR-based recipe import, image upload and PDF export**.  
It focuses on clean architecture, structured data handling and realistic portfolio use cases.

**Stack:** React · Spring Boot · JPA · H2 · PDFBox · Tesseract  
**Status:** Active development
